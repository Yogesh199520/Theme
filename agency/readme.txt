CSS
You will get all used CSS files for theme inside wp-content/themes/PEELAA/css. You can do changes in them according to your need. If you want to add some css, we recommend to add them in style.css, available inside CSS folder


Javascript
You will get all used CSS files for theme inside wp-content/themes/PEELAA/js. You can do changes in them according to your need. If you want to add some custom js, we recommend to add them in custom.js, available inside JS folder


Font
We are using Google font "Poppins" by default. To change the embedded font, look for the tag below in the head section of your website.
<link href='https://fonts.googleapis.com/css?family=Poppins:200,300,400,500,600,700,700i,800' rel='stylesheet' type='text/css' >
